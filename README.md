# Korotoa Frontend OrangeBD

Korotoa Frontend OrangeBD is a front-end application for Korotoa workflow screens.

## Features

- Front-end UI for Korotoa workflows
- API integration readiness
- Responsive page/component structure
- OrangeBD-related project setup

## Modules

- View module: pages, navigation, and user flows
- Component module: shared UI components
- API module: backend request helpers and response handling
- Config module: environment values and service URLs
- Asset module: styles, images, and static resources

## System Architecture

The project acts as the browser/client layer. Components render workflow screens, service helpers call backend APIs, and configuration controls environment-specific endpoints. Backend systems own persistence and business rules.

## Getting Started

```bash
git clone https://github.com/NahinAhmed28/korotoa-frontend-orangebd.git
cd korotoa-frontend-orangebd
npm install
npm run dev
```
