# Korotoa Frontend OrangeBD

Korotoa Frontend OrangeBD is a front-end project for the Korotoa workflow. It appears to be an OrangeBD-associated user interface layer for a larger application.

## Project Details

- Repository: `NahinAhmed28/korotoa-frontend-orangebd`
- Default branch: `master`
- Visibility: public
- Project type: front-end web application
- Main focus: Korotoa UI workflow

## Getting Started

```bash
git clone https://github.com/NahinAhmed28/korotoa-frontend-orangebd.git
cd korotoa-frontend-orangebd
npm install
npm run dev
```

Check `package.json` for exact scripts and configure API base URLs before running locally.
